## Prerequisites

- NodeJS (v18.12.x)
- git

## How to Run

Navigate to the themes folder in your ghost installation.

```bash
$ cd path/to/ghost/content/themes
```

Next, clone the repo

```bash
$ git clone https://github.com/J4yJ3ff/porto.git
```

Install the required packages:

```bash
$ npm install
```

Run the build script:

```bash
$ npm run start
```
